For compiling the code:
gcc project2.c -lm

For executing the code:
./a.out <input >ouput

where, input and output are the names of the input file and output file

The input file must contain:
No of points n in the first line
Penalty c for each line in the second line
the next n lines contain the points in arbitrary order in x y form in different lines

E.g. for input:
5
2
1 1
2 2
4 3
6 1
5 2
